How to Upload Photos to Your Site

Step-by-Step Guide:

  1. Create an "images" folder in the same location as your index.html file
  2. Add your own photos (JPG, PNG, etc.) to the images folder
  3. Edit the photos array in the JavaScript code below (around line 330)
  4. Use this format for each photo you add:
{
    id: 7, // Change this number
    src: 'images/YOUR-PHOTO-FILENAME.jpg', // Your image path
    title: 'Your Photo Title',
    date: '2023-11-15', // YYYY-MM-DD format
    description: 'Your description here...'
    // Tags 已移除，界面更简洁
}

Important: After updating the photos array, save the file and refresh your browser to see the changes.
